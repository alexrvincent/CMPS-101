README file for intstack ADT, modified for decToBinary program.

This directory contains the following files:

README.txt 	The documentation file
decToBinary.c   The source code for a small program that implements the
		intstack ADT as a decimal to binary conversion.
intstack.h   	the header file describing the ADT operations
intstack.c   	The implementation of the intstack ADT
intstackdr.c    A simple program that tests the ADT operations
Makefile	A makefile that compiles the module and driver,
		creating the decToBinary executable and .object
		files for the files needed.
myinclude.h	David Helmbold's personal header file that includes
		commonly used headers and macros.
